## About The Project
너에게 닿기를(Keep in touch)

### 프로젝트 비전
- 더 나은 소통을 통해 더 깊고 진실한 인간관계를 만드는 세상
- 사람들이 서로의 감정과 생각을 편안하게 공유할 수 있는 플랫폼을 제공하여 관계의 질을 향상시킨다

### 핵심 가치
- 진정성: 솔직하고 진실된 소통
- 용기: 어려운 말도 할 수 있는 용기
- 이해: 서로를 더 깊이 이해하려는 노력
- 성장: 피드백을 통한 개인과 관계의 성장
